## features

- animated 2D robot face with expressions + lip sync
- voice: speak to NEXUS, it speaks back
- pick Claude, GPT, or Ollama
- Level 0 — hit the AI, built in
- works offline with local Ollama server on your network
- native iOS feel, no web wrapper

---

## build

requires Xcode 16+ and iOS 17+ target.

```bash
git clone https://github.com/senderpa/4ma-ios.git
open 4ma-ios/FourMA.xcodeproj
```

hit run.
